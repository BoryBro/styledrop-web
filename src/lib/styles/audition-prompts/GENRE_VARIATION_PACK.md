# Audition Still-Cut Genre Variation Pack

Usage:
- Keep the user's existing long-form base prompt as the master prompt.
- Append exactly one block below to the end of that base prompt.
- These blocks are designed to change genre direction only.
- Identity, face, hair, clothing, realism, and strict rules should remain governed by the base prompt.
